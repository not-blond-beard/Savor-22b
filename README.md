## Intro
- 5:5 턴제 전투 게임 레포지터리입니다.
- 슈퍼 푸드를 만들기 위해 랜덤한 확률의 종자를 수확하고,
- 음식을 조합하고, 발전시켜 최고의 슈퍼 푸드를 키워가는 것이 목표!
